Flexible Box and Grid Layout Demo

Flexible Box Layout. The above dotted gray rectangle <div> is set to “display: flex;” so that the colored squares arrange themselves within the container, depending on the screen width. (Try varying the browser window width to see that the layout is partially responsive.) Subsidiary styles “justify-content” (horizontal) and “align-content” (vertical) are set to “center.”

Grid Layout. The above dotted gray rectangle <div> is set to “display: grid;” with 3 columns. Again subsidiary styles “justify-content” (horizontal) and “align-content” (vertical) are set to “center.”
