Hitchhiker’s Guide to a Grid Catalog

This handout shows how to set up a catalog page with 2 columns and 4 rows using the grid layout (display: grid). The page, diagrammed at left, is constructed from a “container” div (gray dotted outline) and 6 colored divs representing the catalog items. The size of each div is implied by their positions, as determined by the “grid-area” style:
• first number is starting row
• second number is starting column
• third number is top edge of ending row
• fourth number is left edge of ending column
