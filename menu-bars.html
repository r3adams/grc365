    Above are three examples of menus made with various tags. The :hover effects provide user feedback. Note that the Home button is not linked because we are already on the Home page.

    Plain Text Menu. This menu is made from <p> tags in a <header> enclosure set to “width: 100%;” “display: flex;” and “justify-content: space-around;” which spaces the links evenly across the page. An orange drop-shadow on :hover provides user feedback.

    Menu Bar. For this menu, “list-item” <li> tags, normally used for bullet points <ul> and numbered lists <ol>, have been “drafted” to help out by making a menu bar. The default block-level li’s are set to “float: left;” to arrange themselves horizontally.

    Button Bar. This menu is similar to the Menu Bar above except that the <li>’s have been set to “border-radius: 50%” to form circles.
