Portfolio

This portfolio page was built as a fixed-width page with an inner “textbox” (light blue dotted line) set to “display: flex;” so that future thumbnails when added will arrange themselves on the page.

The thumbnails can be linked to a web page, a larger version of a photo or drawing, or a downloadable PDF.

The two thumbnails scale up on hover, the first by enlarging overall and the second by enlarging within its fixed-size container div. Use the browser’s “web inspector” to see the code.
